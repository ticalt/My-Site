Skills
HTML - Основа веб-страниц. Определяет структуру контента (заголовки, текст, изображения, кнопки).
CSS - Стилизация веб-страниц. Отвечает за внешний вид элементов (цвета, шрифты, размеры, анимации).
SCSS/SASS - Расширение CSS. Позволяет использовать переменные, вложенности, функции, что упрощает и ускоряет работу с большими стилями.
JavaScript - Язык программирования для добавления интерактивности на веб-страницы (кнопки, формы, анимации).
Bootstrap - Популярный CSS-фреймворк. Предоставляет готовые компоненты (кнопки, сетки, модальные окна) и систему адаптивной верстки.
React - Библиотека JS для создания интерфейсов. Упрощает создание динамических приложений с повторно используемыми компонентами.
AntDesign - Библиотека готовых компонентов для React с современным дизайном. Ускоряет разработку интерфейсов.
NodeJS - Среда для выполнения JS на сервере. Позволяет создавать серверные приложения, взаимодействовать с базами данных.
GIT - Система контроля версий. Позволяет отслеживать изменения в коде, работать в команде и управлять проектами..
MongoDB - NoSQL база данных. Хранит данные в формате JSON (гибкий и масштабируемый способ хранения данных).
TypeScript - Надстройка над JS. Добавляет строгую типизацию, что помогает избежать ошибок и упростить разработку.
